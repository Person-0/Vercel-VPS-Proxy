# **VPS Routes**

This private repo contains routes for my vps so that i do not have to expose the vps's ip to the internet.<br>
APIs and stuff hosted on my VPS is accessed through this expressjs project.

## Steps to run

- install deps
```
pnpm i
```

- run dev to test locally
```
pnpm run dev
```

<br>

# DO NOT FORGET IT IS `pnpm` NOT `npm` else lockfile will become invalid/frozen
